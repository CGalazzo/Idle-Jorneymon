# Idle Jorneymon

Protótipo de um idle game para navegador inspirado no universo Pokémon. O parceiro explora, encontra Pokémon selvagens, batalha e evolui automaticamente.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Produção

```bash
npm run build
```

O progresso é salvo automaticamente no `localStorage` do navegador, com estrutura versionada para futuras migrações.

## Versão atual

`v0.4.0` — sistema completo de progressão com 10 dificuldades, 100 rotas, encontros crescentes, Mini Bosses, Bosses Finais e cenários visuais próprios para cada ambiente.
