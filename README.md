# Idle Jorneymon

Protótipo de um idle game para navegador inspirado no universo Pokémon. O parceiro explora, encontra Pokémon selvagens, batalha e evolui automaticamente.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Produção

```bash
npm run build
```

O progresso é salvo automaticamente no `localStorage` do navegador, com estrutura versionada para futuras migrações.

## Versão atual

`v0.6.1` — sprites transparentes sem recorte durante a exploração, contêineres com espaço extra para Pokémon grandes, adversários persistentes no DOM, aproximação animada por CSS e pré-carregamento completo das imagens da rota para reduzir travamentos.
