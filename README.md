# Idle Jorneymon

Protótipo de um idle game para navegador inspirado no universo Pokémon. O parceiro explora, encontra Pokémon selvagens, batalha e evolui automaticamente.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Produção

```bash
npm run build
```

O progresso é salvo automaticamente no `localStorage` do navegador, com estrutura versionada para futuras migrações.

## Versão atual

`v0.6.0` — encontros pré-carregados e mais fluidos, tamanhos proporcionais das sprites na exploração, evolução automática por nível, decisão de captura com limite de 5 segundos e painel de progresso reposicionado ao lado das informações do chefe.
